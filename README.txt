📋 CONTENIDO:
- index.html        → Página de inicio
- servicios.html    → Página de servicios
- equipo.html       → Página de equipo médico
- agendar.html      → Página para agendar citas
- contacto.html     → Página de contacto
- styles.css        → Estilos personalizados
- script.js         → JavaScript (menú móvil y formularios)

================================================================================
🚀 CÓMO USAR:
================================================================================

1. EXTRAER EL ZIP
   - Descomprime el archivo en cualquier carpeta de tu computadora

2. ABRIR EN EL NAVEGADOR
   - Haz doble clic en "index.html" 
   - O haz clic derecho → "Abrir con" → Tu navegador preferido
   - También puedes arrastrar el archivo al navegador

3. NAVEGAR
   - Usa el menú superior para moverte entre páginas
   - Todos los enlaces funcionan localmente
   - Los formularios muestran alertas al enviar (simulado)

📝 PERSONALIZACIÓN:
================================================================================

CAMBIAR TEXTOS:
- Abre cualquier archivo .html en un editor de texto
- Busca el texto que quieras cambiar
- Guarda y recarga en el navegador

CAMBIAR ESTILOS:
- Abre "styles.css"
- Modifica colores, tamaños, etc.
- Guarda y recarga en el navegador

CAMBIAR TELÉFONO WHATSAPP:
- Busca "wa.me/56912345678" en todos los archivos HTML
- Reemplaza con tu número (formato: código país + número sin espacios)
- Ejemplo: wa.me/56987654321

CAMBIAR URL MEDILINK:
- Abre "script.js"
- Busca "medilink.cl/clinica-consistorial"
- Cambia por tu URL
- O edita directamente en agendar.html
